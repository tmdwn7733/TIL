# CLI
Command Line Interface

커맨드(명령어)를 통해 작동하는 인터페이스

<-> GUI(Graphic User Interface, 보통의 프로그램)


## ❗️리눅스(Linux)의 기본 명령어
* pwd : 현재 폴더의 경로, 위치 확인
* ls : 폴더난 파일의 목록을 출력(list)
* cd : 폴더에 진입
    * cd .. : 상위 폴더
    * cd .: 현재 폴더로 이동
* open . : 현재 폴더를 파일 탐색기로 열기
* mkdir : 새로운 폴더를 생성
* touch : 새로운 파일을 생성
* cat : 파일의 내용을 출력
* rm : 폴더나 파일을 삭제
  * rm -r [폴더명] : recursively(재귀적으로) 폴더를 삭제
* mv : 폴더나 파일의 이름을 변경
  * mv [파일/폴더명] [위치] : 파일 또는 폴더를 이동
* cp : 폴더의 파일을 복사
  * cp -r : 폴더를 복사
* sudo : 관리자 권한으로 다음 명령어를 실행

## ❗️명령어에 등장하는 키워드
* / : 루트 디렉토리
* ~ : 홈 디렉토리(git bash 처음 열면 나오는 기본 폴더) 

## ❗️절대 경로와 상대 경로의 차이
* 절대 경로의 시작은 루트 디렉토리(/)
* 상대 경로의 시작은 현재 디렉토리(.)

## ❗️CLI 파일 편집기
* CLI에서 텍스트 에디터 nano, vim 등을 이용해 파일을 수정하고, 저장
* nano는 기본적으로 다 설치되어있다. 프롬프트에 nano 를 입력하여 실행시킨다.
